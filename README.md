# Grupo Sabores Brutal

Web corporativa en React/Vite para presentar el grupo, sus locales actuales y la proxima apertura Ke Brutal.

## Desarrollo local

```bash
npm install
npm run dev
```

La web se abre en `http://127.0.0.1:5173`.

## Produccion

```bash
npm run build
```

El resultado queda en `dist/`, listo para subir a un hosting estatico.
